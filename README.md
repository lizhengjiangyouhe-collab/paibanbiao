# 档期管家 - 排班与婚礼档期管理

专为婚礼从业者设计的排班与档期管理工具，支持上2休2的工作制度，帮助您轻松管理工作与生活平衡。

## 功能特性

### 📅 智能排班
- 自动计算上2休2的排班
- 支持设置基准日期和班次类型
- 显示农历日期

### 💒 婚礼档期管理
- 标记婚礼日期
- 支持多个地点选择（秀太、喜事汇、其他）
- 记录详细备注信息
- 冲突检测与解决

### 🔍 智能搜索
- 搜索备注内容
- 搜索日期
- 搜索地点
- 点击搜索结果快速跳转

### 🏖️ 公休管理
- 设置年度公休总天数
- 自动统计已用和剩余公休
- 公休抵扣冲突日期

### 📱 iOS优化
- 毛玻璃效果设计
- 底部滑出式弹窗
- 适配刘海屏和底部黑条
- 支持添加到主屏幕

### 📦 数据安全
- 固定存储Key，防止版本更新导致数据丢失
- 数据保险箱保护，有老数据时绝不覆盖
- 旧数据自动迁移到新格式

### 💾 备份与恢复
- 导出所有数据为JSON文件
- 从备份文件恢复数据
- 支持跨设备数据迁移

### 🌐 PWA离线支持
- Service Worker缓存机制
- 离线也能正常使用
- 网络优先策略，有网时更新

## 使用说明

### 首次使用
1. 点击右上角设置按钮
2. 设置基准日期（任意一天）
3. 选择基准日期的班次类型
4. 设置年度公休总天数（可选）

### 添加婚礼档期
1. 点击日历上的日期
2. 打开"婚礼档期"开关
3. 选择地点
4. 添加备注（可选）
5. 如果是工作日，设置替班或公休

### 搜索功能
1. 在搜索框输入关键词
2. 查看匹配的搜索结果
3. 点击结果跳转对应日期

## 技术栈

- React 18
- Tailwind CSS
- Lunar-JavaScript（农历）
- LocalStorage（数据存储）

## 部署说明

### GitHub Pages部署
1. 将此仓库部署到GitHub
2. 在仓库设置中启用GitHub Pages
3. 选择 `main` 分支作为源
4. 等待部署完成

### 添加到iPhone主屏幕
1. 在Safari中打开部署的网址
2. 点击分享按钮
3. 选择"添加到主屏幕"
4. 确认添加

## 数据存储

所有数据都存储在浏览器的LocalStorage中，包括：
- 排班基准设置
- 婚礼档期记录
- 公休天数设置

数据不会上传到任何服务器，完全私密。
